----------------------------------------

Defaults
    If any field is left blank:
        Use neutral, realistic values.
        Use attire appropriate to the context of the pose or background.
        Use a friendly, focused expression.
        Use straight-on, eye-level framing.

----------------------------------------

Rules
    Character details override style when they conflict.
    Do not alter identity across poses.
    Do not change attire unless explicitly instructed.
    Keep anatomy realistic unless the chosen style requires stylization.

----------------------------------------


How to Use Occupation Presets
    These presets are optional.
    Use them only if you want to quickly define attire and accessories for a common eLearning role.
    If you choose a preset:
        It will pre-fill attire, accessories, and a few identity cues.
        You may edit any field after applying the preset.
    If you do not choose a preset:
        Fill in the Character Details fields manually.
        Defaults will apply wherever fields are left blank.
