Backgrounds
Version 0.2 — March 2026

----------------------------------------

Purpose
    Define the environment or setting behind the character.
    Backgrounds should be style‑agnostic and pose‑agnostic.
    They provide context, not narrative.

----------------------------------------

Background Reference Image (Optional)
    If the user uploads a background reference image:
        Use it as the primary source for environment, lighting, perspective, and spatial layout.
        Match the overall look, depth, and composition of the uploaded image.
        Bypass the text-based background module.
        Do not reinterpret or alter the environment unless explicitly instructed.
    If no background reference image is provided:
        Use the selected background from the Backgrounds list.

----------------------------------------
Background Description (Optional)
    If the user provides a written description of the background:
        Use the description as the primary source for environment, lighting, perspective, and spatial layout.
        Interpret the description literally and avoid adding elements not mentioned.
        Bypass the text-based background module.
    If both a description and a background reference image are provided:
        The background reference image takes priority.

Defaults
    If background is unspecified:
        Use Neutral — Soft Gradient.
    If lighting is unspecified:
        Use soft, even lighting appropriate to the environment.
    If depth is unspecified:
        Use shallow depth of field for clarity.

----------------------------------------

Rules
    Backgrounds must not alter character identity.
    Backgrounds must not override pose or attire.
    Backgrounds should remain contextually appropriate to the character’s role.
    Avoid clutter unless explicitly requested.
    Keep perspective consistent with the chosen pose orientation.
    When a background reference image is provided, it overrides all text-based background selections.
