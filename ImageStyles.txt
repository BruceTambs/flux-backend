Clean Modern Illustration
    Description
        A polished, professional illustration style with smooth shapes and balanced detail.
    Rendering rules
        Use clean edges, simplified forms, and controlled shading.
    Lighting rules
        Use soft, even lighting with minimal contrast.
    Color rules
        Use a neutral, modern palette with subtle gradients.
    Line/texture rules
        Use thin, consistent linework or no visible lines.
    Tags
        clean, modern, professional

Semi-Realistic Portrait
    Description

    Rendering rules

    Lighting rules

    Color rules
        Use realistic skin tones and controlled saturation.
    Line/texture rules
        Use minimal linework; rely on shading for form.

Painterly Digital

    Color rules

    Line/texture rules

Flat Vector
    Description
        A crisp, geometric vector style with no shading.
    Rendering rules
        Use flat shapes, clean geometry, and no gradients.
    Lighting rules

    Color rules

    Line/texture rules
        Use uniform stroke weight or no outlines.
    Tags
        vector, flat, geometric

Photorealistic Construction
    Description
        A realistic, documentary-style construction environment with accurate materials, lighting, and safety gear.
    Rendering rules
        Use true-to-life proportions, natural skin textures, and authentic construction equipment and PPE. Preserve realistic fabric folds, tool details, and environmental grit.
    Lighting rules
        Use natural outdoor or indoor worksite lighting with soft shadows and realistic highlights. Match the directional light typical of open construction zones.
    Color rules
        Use accurate, high-visibility safety colors (orange, yellow, reflective silver) and natural tones for skin, clothing, and materials. Maintain realistic saturation and contrast.
    Line/texture rules
        No visible linework. Use photographic texture detail for clothing, tools, machinery, and surfaces.
    Tags
        photorealistic, construction, safety-gear, documentary

Retro Pulp Comic (1950s Sci-Fi)
    Description
        A vintage mid‑century sci‑fi comic style with bold colors, dramatic poses, and classic pulp adventure energy.
    Rendering rules
        Use exaggerated action, expressive character gestures, and dynamic compositions. Include retro space suits, ray guns, rockets, and alien landscapes when relevant.
    Lighting rules
        Use high‑contrast lighting with sharp highlights and stylized shadow shapes typical of vintage comic covers.
    Color rules
        Use bold, saturated primary colors with limited palettes. Emphasize flat color blocks, simple gradients, and retro print hues.
    Line/texture rules
        Use strong black ink outlines, varied line weight, and subtle halftone or print‑texture effects. Preserve the look of classic comic printing.
    Tags
        retro, pulp, comic, sci-fi, vintage, inked
